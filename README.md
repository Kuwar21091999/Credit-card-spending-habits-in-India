##  Project Overview
This project analyzes real-world credit card transaction data in India using SQL.  
The goal is to understand customer spending behavior, identify high-value transactions, and extract meaningful business insights from raw transactional data.

---

##  Objectives

- Analyze total spending by category and city  
- Identify top spending customers  
- Calculate average transaction amount  
- Understand gender-wise spending patterns  
- Extract high-value transactions  

---

##  Tools & Technologies Used

- SQL (MySQL / SQL Server)
- Aggregate Functions (SUM, AVG, COUNT)
- GROUP BY
- ORDER BY
- WHERE & HAVING Clauses
- Subqueries

---

##  Dataset Information

The dataset contains:

- Customer ID  
- Transaction Amount  
- City  
- Gender  
- Card Type  
- Spending Category  
- Transaction Date  

---

##  Key Analysis Performed

1. Calculated total transaction amount across cities.
2. Identified top 5 highest spending categories.
3. Computed average transaction value.
4. Filtered high-value transactions using WHERE clause.
5. Used GROUP BY to analyze category-wise spending.
6. Applied aggregate functions for business insights.

---

##  Key Insights

- Certain cities contribute significantly to overall credit card spending.
- Spending behavior varies across categories and gender.
- A small group of customers contributes a large portion of revenue.
- High-value transactions impact overall revenue trends.
